# Biblioteca-PO
Biblioteca feita durante aula de desenvolvimento de sistemas comandada pelo professor Salomão destinada a programação orientada a objeto

# Como clonar o projeto

1. Abrir o Github;
2. Acessar repositório
3. Copiar link
4. Abrir terminal (CMD, PowerShell, GitBash) na pasta desejada
5. Digitar o comando "git clone [Link repositório]"
6. Digitar no terminal "code" [nome da pasta]
